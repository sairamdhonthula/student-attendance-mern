A professional MERN stack-based Student Attendance Management System featuring secure JWT authentication, dynamic attendance tracking, real-time analytics dashboard, and Excel export functionality. Designed for educational institutions to streamline attendance processes with modern UI and scalable backend architecture.

The Student Attendance Management System is a full-stack web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js) that enables teachers to securely log in, mark daily student attendance, and analyze attendance performance through interactive dashboards and reports.

This system replaces manual attendance processes with a digital, efficient, and scalable solution designed for educational institutions.
